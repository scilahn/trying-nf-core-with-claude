# Project Overview
The purpose of this project is to learn how to use Claude Code (and agentic coding in general) to write agent assisted nf-core (nf-co.re) pipelines and prepare them for execution on the seqera platform.

## Architecture
- Write pipelines to /source
- Pipelines from nf-core are available at https://nf-co.re/pipelines.

## Execution
- Only write the nextflow scripts
- DO NOT RUN THE pipelines
